Стор прнимает и работает с данными в двух структурах:
DSL-like языковая структура
```
Объект
  Свойство: Значение1
  Свойство: Значение2
  Модификатор
    Свойство: Значение3
  Свойство: Значение4
  Модификатор
    Модификатор
      Свойство: Значение5
  Модификатор
    Свойство: Значение6
  Свойство: Значение7
```

Либо Json структура для обработки
```
{
    object: [
        {property: ["val1"]},
        {property: ["val3"]},
        {modificators: [
            {property: ["val2"]}
        ]}
        {property: ["val4"]},
    ]
    }
}
```
